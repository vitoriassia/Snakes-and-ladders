# Cobras e escadas!

### Como rodar?

 1-> Instalar o <a href="https://flutter.dev/docs/get-started/install">Flutter</a> na sua máquina.

 2-> E rodar o comando abaixo no diretório do projeto:
 
 # Modo DEBUG
 ```console

foo@terminal:~$ flutter run

```
# Modo RELEASE
```console

foo@terminal:~$ flutter run --release

```
